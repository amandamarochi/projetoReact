## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/amandamarochi/ProjetoReact.git

# Access
$ cd ProjetoReact

# Install dependencies
$ yarn

# Run the project
$ yarn dev

# The server will initialize in the <http://localhost:3000>
```
